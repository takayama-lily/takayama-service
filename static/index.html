Usage:

查询战绩：

  • 调用 [/api?m=searchAccountByPattern&pattern=xxxxxx] pattern可以是8位数字id或昵称
    try: /api?m=searchAccountByPattern&pattern=天才麻将杏杏
  • 如果传入的是8位数字会拿到一个decode_id；如果是昵称会拿到一个match_accounts，取里面第一个id
  • 传入上面的id，调用 [/api?m=fetchAccountInfo&account_id=xxxxxx] 获得个人基本信息
    try: /api?m=fetchAccountInfo&account_id=33336
  • 传入上面的id，调用 [/api?m=fetchAccountStatisticInfo&account_id=xxxxxx] 获得个人详细战绩(段位,友人,比赛)
    try: /api?m=fetchAccountStatisticInfo&account_id=33336

查询牌谱：

  • 调用 [/api?m=fetchGameRecord&game_uuid=xxxxxx] 传入牌谱id得到牌谱数据
  • 调用 [/record?id=xxxxxx] 传入牌谱id得到牌谱详细战斗数据
    try: /record?id=191108-34030ae3-6e82-472e-b5c1-a43f2aaa8827

查询大会室牌谱：

  • 调用 [/api?m=fetchCustomizedContestByContestId&contest_id=xxxxxx] 传入大会室id得到unique_id
  • 调用 [/api?m=fetchCustomizedContestGameRecords&unique_id=xxxxxx] 传入unique_id得到牌谱列表，一次最多20条
  • 调用 [/api?m=fetchCustomizedContestGameRecords&unique_id=xxxxxx&last_index=xxxx] 传入last_index可以往前翻页
